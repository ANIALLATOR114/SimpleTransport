About SimplyTransport

SimplyTransport is a project developed to simplify transit data lookup. It aims to provide an accessible and efficient way to access transport data provided by Transport For Ireland in the GTFS format.

This website is hosted, developed, maintained and provided at no cost. It is not officially affiliated with any government or transport authority. The accuracy or inaccuracy of any timing data is entirely at the control of the data providers.

Features

SimplyTransport is constantly growing, with new features being added regularly. It provides an API for accessing formatted GTFS data, with the ability to query specific subsets of data. This functionality is not available in the limited government APIs. Additional data models have also been defined to make working with the datasets more accessible.

Some key features include:

  • Searching routes / stops
  • Viewing static schedules / trips
  • Always updating Realtime ETAs
  • Maps of routes and stops
  • Stop Features per stop ( shelter, realtime display, bins etc )
  • Open API spec for most datasets

Technology

SimplyTransport is built using a variety of technologies to ensure efficiency and reliability. Further technical details are available on the GitHub repository.

  • Python
  • Litestar (web framework)
  • PostgreSQL
  • Redis
  • Docker
  • NginX
  • CloudFlare

Contribute

Contributions to SimplyTransport are always welcome. If you have any suggestions or want to report an issue, please feel free to open an issue on the GitHub repository. Instructions on how to run the project locally are also available in the README.
