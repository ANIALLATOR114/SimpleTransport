About SimplyTransport

SimplyTransport is a project developed to simplify transit data lookup. It aims to provide an accessible and efficient way to access transport data provided by Transport For Ireland in the GTFS format.

Features

SimplyTransport is constantly growing, with new features being added regularly. It provides an API for accessing formatted GTFS data, with the ability to query specific subsets of data. This functionality is not available in the limited government APIs. Additional data models have also been defined to make working with the datasets more accessible.

Technology

SimplyTransport is built using a variety of technologies to ensure efficiency and reliability:

  • Python
  • Litestar (web framework)
  • PostgreSQL
  • Redis
  • Docker
  • NginX
  • CloudFlare

Contribute

Contributions to SimplyTransport are always welcome. If you have any suggestions or want to report an issue, please feel free to open an issue on the GitHub repository. More technical explanations of the data can be found on the GitHub repository too as well as instructions on how to run the project.
