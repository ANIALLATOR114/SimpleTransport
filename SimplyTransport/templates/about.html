About SimplyTransport

SimplyTransport is a web application designed to provide comprehensive and comprehensible Transport Information in Ireland.

The website and backend are open-source at SimplyTransport

This project has been solely developed by Niall

As development is on-going, new featues are being added all the time. If you have any suggestions, please feel free to open an issue on the GitHub repository. It's possible it may already be planned, but if not, I'd love to hear about the use-case!

Transport Data

The data used by SimplyTransport is provided by Transport For Ireland in the GTFS format.

More technical explanations of the data can be found on the GitHub repository.

API

SimplyTransport provides an API for accessing formatted GTFS data with the ability to query specific sub-sets of data. The limited government APIs do not have this functionality and when I asked them about it they informaed me it wasn't coming any time soon.

I have also added additional models of data I have defined to make working with the datasets more acessible. For example, the API provides a list of all stops in a route, which is not available in the GTFS data, and much more.

Technology

SimplyTransport is built using the following technologies:

  • Python
  • Litestar (web framework)
  • PostgreSQL
  • Redis
  • Docker
  • NginX
  • CloudFlare
