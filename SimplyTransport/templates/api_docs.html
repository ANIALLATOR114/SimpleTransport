API Documentation

The SimplyTransport API docs are available under multiple schemas.

  • ReDoc - A simple, clean, responsive documentation page.
  • Swagger UI - A more complex, but feature-rich documentation page.
  • StopLight - Interactive API Docs with embeddable web components.
  • OpenAPI JSON - The raw JSON used to generate the documentation.
  • OpenAPI YAML - The raw YAML file used to generate the documentation.
